# data/raw

This folder contains all data used in the reef interventions analysis. Files here are never modified by hand.

`reef-interventions_clean.csv` is the analysis-ready dataset generated by the private script `R/00_download_and_clean.R` and its associated custom functions in `R/functions/` — neither of which are included in this repository. All other analysis scripts use this file as their starting point.

---

## Files

### `reef-interventions_clean.csv`

The cleaned and processed intervention review database. Downloaded from the master Google Sheets database, filtered, formatted, and enriched with contextual predictors. Each row represents one outcome, from one study, at one site.

#### Column structure

Columns follow a prefix convention:

| Prefix | Category | Description |
|--------|----------|-------------|
| `M_` | Metadata | Bibliographic information: authors, title, year, DOI, journal |
| `I_` | Intervention | Strategy, type, duration, and design of the management intervention |
| `O_` | Outcome | Indicator, category, and direction of impact on nature / people / NCP |
| `P_` | Predictor | Contextual variables: biophysical, governance, socio-economic, anthropogenic stressors |

#### Key fields

| Column | Description |
|--------|-------------|
| `row_id` | Unique row identifier |
| `study_id` | Unique identifier for each study |
| `citation` | Formatted bibliographic reference |
| `intervention_full_name` | Full intervention name as recorded in the database |
| `intervention_simple_name` | Simplified intervention label (dimension - category) |
| `intervention_dim` | Intervention strategy (e.g. FishMan, MarCon, SocioDev) |
| `intervention_cat` | Intervention type within strategy |
| `outcome_catdim` | Outcome dimension and category combined |
| `outcome_dim` | Outcome dimension (Biodiversity, Ecosystem services, Human well-being) |
| `outcome_cat` | Outcome domain (e.g. Species, Fishery, Governance) |
| `outcome_impact` | Direction of impact: positive, negative, neutral, mixed |
| `lat` / `lon` | GPS coordinates of the study site |
| `country` / `region` | Study location |
| `study_year` | Year of publication |
| `data_year` | Year the outcome data were collected |
| `age` | Age of the intervention at time of data collection (years) |
| `record_weight` | Inverse-frequency weight to reduce over-represented interventions |

### `predictors/`

External predictor datasets used to enrich the review database. These are loaded by `R/00_download_and_clean.R` and matched to study sites by GPS coordinates.

| File | Source | Description |
|------|--------|-------------|
| `allreefs_WGS84.shp` | Andrello et al. (2022) *Conservation Letters* | Anthropogenic threat indicators at reef polygons |
| `hdi_raster_predictions.tiff` | MOSAIKS (mosaiks.org) | Downscaled Human Development Index (0.1° resolution) |
| `human_dependence_marine_ecosystems.csv` | Selig et al. (2018) | Nutritional and economic dependence on marine ecosystems |
| `HDR21-22_Statistical_Annex_GII_Table.csv` | UNDP | Gender Inequality Index (2021) |
| `HDR21-22_Statistical_Annex_HDI_Table_2.csv` | UNDP | Human Development Index (2021) |
| `API_SI.POV.GINI_DS2_en_excel_v2.csv` | World Bank | Gini coefficient (income inequality) |
| `WGI_clean.csv` | World Bank | Worldwide Governance Indicators (2021) |
| `NOAA/yearly_max_DHW/` | NOAA | Yearly maximum degree heating weeks (1985–2023, .tif rasters) |
