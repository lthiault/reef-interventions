# docs

This folder contains supplementary data files associated with Paper 1. Each file is a standalone Excel spreadsheet documenting a key component of the systematic review methodology.

---

## Files

### `Table_S1_Interventions.xlsx`

Definitions, specific selection criteria, and partial search strings for all management interventions analyzed in the systematic review. One row per intervention, organized by management strategy.

| Column | Description |
|--------|-------------|
| `Management strategy` | Top-level intervention category (e.g. Area-based conservation, Fisheries management) |
| `Management intervention` | Specific intervention name |
| `Description` | Full definition of the intervention |
| `Examples, alternative names` | Common synonyms and examples used in the literature |
| `Specific criteria` | Inclusion and exclusion criteria applied during screening |
| `Search date (n results)` | Date the literature search was conducted and number of results returned |
| `Partial search strings` | Boolean search string used in the systematic literature search |

---

### `Table_S2_Outcomes.xlsx`

Definitions and examples for the 15 outcome domains analyzed in the systematic review, spanning biodiversity, ecosystem services, and human well-being.

| Column | Description |
|--------|-------------|
| `Outcome Dimension` | Top-level dimension: Biodiversity, Ecosystem services, or Human well-being |
| `Outcome Domain` | Specific outcome domain (e.g. Species, Fishery, Governance) |
| `Description` | Definition of the outcome domain |
| `Examples of indicators` | Concrete indicators used to measure this outcome in the literature |

---

### `Table_S3_Contextual_variables.xlsx`

Definitions and sources for all contextual predictor variables extracted for each study site, covering biophysical, socio-economic, governance, and anthropogenic stressor dimensions.
